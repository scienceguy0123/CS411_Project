# CS411_Project
To run back end:
1. cd to server
2. run "npm install" in terminal
3. run "node index.js" in terminal
4. (optional) go to  "http://localhost:3001"

To run front end:
1. cd to bookdeal
2. run "npm install" in terminal
3. run "npm start" in terminal
4. go to "http://localhost:3000"
